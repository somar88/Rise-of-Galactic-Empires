# Galaxy Setup
- Build the Galaxy:
	- The Galaxy is built by all players randomly generating the map, using 
		- [Star Systems](../pieces/space/star-system.md) Hex Tiles
		- [Space Anomaly](../pieces/space/space-anomaly.md) Hex Tiles
	- Make a stack of [Anomaly Tiles](../world/anomaly-tiles.md) equal to two times the number of the players
	- Players should draw from a shuffled stack of all [Star Systems](../pieces/space/star-system.md) and [Space Anomaly](../pieces/space/space-anomaly.md)  tiles and build the galaxy starting from their home base location toward the galaxy center.
	- Players are allowed to pick from [Space Anomaly](../pieces/space/space-anomaly.md) hex tiles instead of [Star System](../pieces/space/star-system.md) tile.
	- Players are allowed to place an anomaly tile if not adjacent to any other [Space Anomaly](../pieces/space/space-anomaly.md) tiles(s) and other players homebases (they are allowed to add it adjacent to their own homebase)
