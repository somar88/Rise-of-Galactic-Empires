# Conflict Phase

consists of this three parts:
	1. [Space Battle](../../conflict/space-battle.md): Run space battle for each space conflict
		- start with the conflicts in [players order](../player-order.md)
		- multiple battles can take place at the same time if participating players agree
	2. [Landing](../../conflict/landing.md) Step
		- Players can initiate invasion / landing(s) or ground reinforcement
	3. [Ground Battle](../../conflict/ground-battle.md): run ground battle for each ground conflict
		- start with the conflicts in players order
		- multiple ground battles can take place at the same time if participating players agree
