# Planets

Each [Star System](../space/star-system.md) has multiple planets but only habitable or industrial planets are considered in this game and are explored, colonized, exploited and fought for.

There are multiple types of planets:
- [Moons](moons.md)
- [Planets](planets.md)
- [Giant Planets](giant-planets.md)

Each planet has a card that should be searched for and drawn by the player who comes by this planet when exploring a star system

The drawn planet cards are to be kept faced up next to the galaxy map

When drawing the planet cards from their planet type deck
- Player should execute the events and resources draw part of each card 
- Apply the events is they have when revealed effect ***or***
- Tug the card to the planet so the additional event / effects apply to the planet permanently or according to what the event card says 
- Add resources to the planet card if the resource card says so, ***or*** 
- Tag the card to the planet so the additional resource / effects apply to the planet permanently or according to what the resources card says
