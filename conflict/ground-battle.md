Ground Battle: can start only if [units](../pieces/ground/ground-forces.md) are on the ground from to empires that are not allied with each other and or NP Empires that are available on the planet
  1. each empire must join with all [ground units](../pieces/ground/ground-forces.md) that are located on the planet
  2. before a [battle](ground-battle.md) round starts, each player decides if they want to retreat / surrender before next battle round
  3. starting with active player, each player applies (role dice, cards, tech powers, etc.) that have before ground battle affect(s). 
  And all happen at same time.
  1. starting with active player, each player roles the amount of dice according of their armies(s) power and assign damage:
    1. each unit, weapon must be assigned a target before rolling its dice
    2. damages are assigned accordingly
    3. this should be done 3 times
    4. after three fire exchange rounds, the battle round ends.
    5. if [ships](../pieces/space/fleets.md) carrying land [troops](../pieces/ground/ground-forces.md) are still available, the empire can try [Landing](landing.md) them and initiate a [Ground Battle](ground-battle.md).
    6. after each battle round is done, starting with the active player, each player applies (role dice, cards, tech powers, etc.) that they have before battle affect(s). All happens at same time.
